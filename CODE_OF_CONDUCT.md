# Code of Conduct

This project follows a simple rule: be direct, respectful, and useful.

## Expected Behavior

- Assume good intent.
- Critique code and ideas, not people.
- Keep issues and pull requests focused on the work.
- Be clear when giving feedback, especially when rejecting an API or design.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language.
- Publishing private information.
- Repeated off-topic disruption.

Maintainers may remove comments, close threads, or block contributors who make
the project harder to work on.
