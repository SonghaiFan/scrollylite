# Security Policy

ScrollyLite is pre-1.0. Security fixes target the latest release line.

## Reporting

Please report security issues privately by opening a GitHub security advisory on
the repository. If advisories are unavailable, contact the maintainer directly
before opening a public issue.

Include:

- affected version or commit
- a minimal reproduction
- expected impact
- any known workaround

Do not publish exploit details until a fix or mitigation is available.
